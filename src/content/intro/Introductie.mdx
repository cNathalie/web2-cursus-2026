## Welkom!

Fijn dat je erbij bent! In deze cursus **Web 2** ontdekken we stap voor stap hoe het internet werkt, wat er achter de schermen van je browser gebeurt en hoe je zelf dingen kunt laten bewegen, reageren en veranderen op een webpagina. Alles op een rustige, haalbare manier — ook als je nog nooit één regel code hebt geschreven.

<img src={WebDevImg} className="btwn-text" />

De cursus bevat **interactieve oefeningen**, die je rechtstreeks in de console van je browser kunt uitvoeren. En nog leuker: _de cursus is zelf een webapp, gebouwd in React_.

## Hoofdstukken

- **Het internet:** hoe clients, servers en HTTP samenwerken.
- **Browser developer tools:** je ingebouwde gereedschapskoffer.
- **HTTPS:** beveiliging van je webverkeer.
- **JavaScript basis:** objecten, events, timers, en _advanced_ DOM‑manipulatie.
- **DOM tree:** de structuur van je webpagina als “boom”.
- **Events:** hoe je kunt reageren op gebruikersacties.
- **CSS manipulatie met JavaScript:** dynamisch stijlen en veranderen.
- **Forms:** gebruikersinvoer verwerken.
- **Timers:** tijdgebonden acties uitvoeren.
- **Advanced DOM:** geavanceerde manipulatie van de DOM.
- **Objecten:** werken met objecten en hun eigenschappen.
- **Fetch API:** data ophalen van het internet.
- **JSON introductie:** eenvoudige uitwisseling van gegevens.
- **Mini intro TypeScript:** eens proeven van typed code.

Je leert telkens **praktisch** werken met voorbeelden en kleine experimenten.

## Werkwijze

We starten eenvoudig en bouwen geleidelijk op. Je hoeft geen programmeerkennis te hebben: nieuwsgierigheid en zin om te proberen volstaan.

Neem gerust een tas koffie of thee, open je browserconsole, en laat ons samen ontdekken hoe je van een webpagina iets maakt dat leeft, beweegt en reageert.

**Vergeet niet: fouten maken mag!** Het is een essentieel onderdeel van leren. Dus als je vastloopt, aarzel niet om hulp te vragen.

In de cursus kan je heel wat code-voorbeelden terugvinden; deze zien er zo uit:

```js
// Dit is een comment in de code
const tekst = "Dit is een stukje code";
console.log(tekst);
```

Je kan de code ook gemakkelijk kopiëren naar je klembord met behulp van het knopje rechtsboven.

Alle code-voorbeelden kan je uitvoeren in de Console van je Browser Developer Tools (zie hoofdstuk).
Dit zorgt ervoor dat je altijd in één oogopslag de theorie én de praktijk op je scherm hebt staan.

Gaandeweg leren we ook in Visual Studio Code werken.

## Evaluatie

Gedurende dit semester werk je de helft van de tijd aan een eigen project. Hoe je dit invult, mag je helemaal zelf kiezen. De enige eis is dat je iets bouwt dat gebruik maakt van de concepten die we in de cursus behandelen. Belangrijk: er wordt niet verwacht dat je gebruik maakt van React of andere frameworks/libraries. De behandelde technologieën uit de cursus (HTML, CSS, JavaScript, DOM-manipulatie, Fetch API, etc.) zijn de focus.

**Beoordeling:** Je project wordt beoordeeld op basis van de functionaliteit, de toepassing van de geleerde concepten en de algehele presentatie.

**Eindpresentatie:** Aan het einde van het semester presenteer je jouw project aan de docent. Dit is een kans om je werk te laten zien, en aan te tonen dat je de behandelde concepten begrijpt en kunt toepassen.

**Wie zijn eigen code niet kan uitleggen tijdens de eindpresentatie krijgt meteen een onvoldoende voor het project en dus voor het hele opleidingsonderdeel.**

## Artificiële Intelligentie

Het gebruik van AI tools zoals ChatGPT is toegestaan voor brainstormen en het genereren van ideeën. Echter, de uiteindelijke code
en implementatie moeten jouw eigen werk zijn. Plagiaat wordt streng bestraft.

<img src={DumbAi} className="btwn-text" />

Hoewel deze technologieën indrukwekkend en handig kunnen zijn, is er een belangrijke kanttekening: Het doel van programmeeroefeningen
is niet alleen om tot een werkende oplossing te komen, maar vooral om zelf te **leren denken** als een programmeur.
Probleemoplossend denken, logisch redeneren en het leren omgaan met fouten zijn essentiële vaardigheden die je enkel ontwikkelt door zelf
actief aan de slag te gaan. Wanneer je AI gebruikt om code voor je te laten schrijven, loop je het risico deze vaardigheden niet of onvoldoende te ontwikkelen.
Op korte termijn lijkt het misschien efficiënt, maar op lange termijn kan het je leerproces en je zelfstandigheid als programmeur belemmeren.

**Wil je je echt onderscheiden als programmeur?** Dan is het belangrijk dat je de fundamenten beheerst. Gebruik AI gerust als hulplijn om uitleg te
krijgen over concepten of foutmeldingen, maar laat het niet je denkwerk overnemen.  
**Prompt-prutsers zijn er al meer dan genoeg.**

Kortom: gebruik je hoofd, niet alleen een tool.
Zo bouw je een stevige basis op waar je later écht iets aan hebt.

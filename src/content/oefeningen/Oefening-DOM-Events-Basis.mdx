## Oefening: Interactieve Fotoalbum

In deze oefening oefen je **elementen opvragen** en **event listeners toepassen** zonder dynamisch elementen toe te voegen of te verwijderen.
Je maakt deze oefening in een nieuw Visual Studio Code projectje.

Je gaat een eenvoudig interactief fotoalbum bouwen waar bezoekers kunnen:

- Door foto's navigeren met knoppen (volgende/vorige)
- Foto's als favoriet aanduiden door op een ster te klikken
- Een beschrijving zien/verbergen met een knop
- Het huidige fotogetal zien (bijv. "Foto 2 van 5")

## Stap 1: HTML-structuur

Maak een bestand `album.html`:

```html
<!DOCTYPE html>
<html lang="nl">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Fotoalbum</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="container">
        <h1>Mijn Fotoalbum</h1>
        
        <!-- Foto-weergave -->
        <div class="photo-frame">
            <img id="photoImage" src="photo1.jpg" alt="Foto">
            <p id="photoCounter">Foto 1 van 5</p>
        </div>
        
        <!-- Foto-titel -->
        <h2 id="photoTitle">Een mooie zonsondergang</h2>
        
        <!-- Beschrijving (verborgen) -->
        <p id="photoDescription" class="hidden">
            Dit is een prachtige foto van een zonsondergang boven het strand. 
            Genomen op een mooie zomeravond met een professionele camera.
        </p>
        
        <!-- Knoppen -->
        <div class="button-group">
            <button id="prevBtn">← Vorige</button>
            <button id="favoriteBtn">★ Favoriet</button>
            <button id="nextBtn">Volgende →</button>
        </div>
        
        <!-- Beschrijving knop -->
        <button id="toggleDescBtn" class="secondary">Beschrijving tonen</button>
    </div>

    <script src="main.js"></script>
</body>
</html>
```

## Stap 3: JavaScript

Maak een bestand `main.js`. **Lees dit eerst goed door en begrijp elke regel!**

### Vereisten

Jouw JavaScript moet het volgende bevatten:

1. **Window load event**: Wacht tot de pagina volledig is geladen
2. **Elementen opvragen**: Haal alle benodigde elementen op met:
   - `document.getElementById()`
   - `document.getElementsByClassName()` (optioneel)
3. **Event listeners**: Voeg event listeners toe aan:
   - De "Vorige" en "Volgende" knoppen (click event)
   - De "Favoriet" knop (click event)
   - De "Beschrijving" knop (click event)
4. **classList manipulatie**: Voeg toe/verwijder classes om:
   - Favoriete status te tonen
   - Beschrijving te tonen/verbergen
5. **textContent**: Update de tekst van:
   - Fotogetal
   - Knoppen tekst
6. **Array gebruiken**: Gebruik een array om de foto-informatie (titel, beschrijving, src) op te slaan 
en te navigeren tussen foto's.

### Foto's
Foto's zonder auteursrechten kan je vinden op https://pixabay.com/nl/. Download de foto's en plaats ze in een map `images` binnen je project.

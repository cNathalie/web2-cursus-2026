## Herhalingsoefening na hoofdstukken 'Fetch API & JSON'

**Doel**: herhaling van fetch API, het verwerken van JSON data, en asynchrone programmering met .then().catch() en async/await.

**Uitvoering**: dit programma werkt met HTML elementen en haalt data op via HTTP requests.
Maak deze opdracht in een nieuw project in VSCode

### Vooraf

- Je hebt een HTML bestand, een CSS bestand en een JavaScript bestand nodig.
- Je kan je project lokaal testen met een live server (VS Code extensie [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)) of een ander development server.
- Voor de externe API request gebruiken we de Star Wars API (gratis, geen API key nodig): `https://swapi.info/api/planets/1`
- Voor de lokale JSON file, maak je een bestand aan in je project (bijvoorbeeld `bibliotheek.json`)

(Je mag ook een andere API of JSON file gebruiken, maar zorg dat deze publiek toegankelijk is.)

### Situatie

Je gaat een webpagina maken die data ophaalt via twee verschillende methodes:

1. **Methode 1 (Promise chaining)**: Data ophalen van een externe public API en weergeven met `.then().catch()`
2. **Methode 2 (Async/await)**: Data ophalen van een lokale JSON file en weergeven met `async/await` en `try/catch`

Elke methode kan via een aparte knop aangeroepen worden, zodat de gebruiker kan kiezen welke data getoond wordt.

### Opdracht

#### 1. Maak een HTML-bestand

Maak een HTML-bestand met:

- Een titel (h1)
- Twee knoppen: één voor "Haal Star Wars data op" en één voor "Haal bibliotheek data op"
- Twee containers (div-elementen) met id's: `starwars-result` en `bibliotheek-result` voor het tonen van resultaten
- Een container voor foutmeldingen: `error-message`
- Link naar je CSS-bestand en JavaScript-bestand

```html
<!DOCTYPE html>
<html lang="nl">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Fetch API & JSON Oefening</title>
    <link rel="stylesheet" href="style.css" />
  </head>
  <body>
    <h1>Fetch API & JSON Oefening</h1>

    <button id="btn-starwars">Haal Star Wars data op</button>
    <button id="btn-bibliotheek">Haal bibliotheek data op</button>

    <div id="error-message" class="error"></div>

    <div id="starwars-result"></div>
    <div id="bibliotheek-result"></div>

    <script src="script.js"></script>
  </body>
</html>
```

#### 4. Methode 1: Promise chaining met .then() en .catch()

Schrijf een functie `haalStarWarsData()` die:

- Gebruikt `fetch()` om data op te halen van: `https://swapi.info/api/planets/1`
- De response omzet naar JSON
- Vervolgens de data verwerkt: naam, aantal inwoners, klimaat, etc. weergeven als HTML
- Een error afhandelt
- Het resultaat toont in het `starwars-result` element

```javascript
const haalStarWarsData = () => {
  // Jouw code hier
  fetch("https://swapi.info/api/planets/1")
    .then //....
    ();
};
```

#### 5. Methode 2: Async/await met try/catch

Schrijf een functie `haalBibliotheekData()` die:

- Async is gedeclareerd
- Gebruikt `fetch()` om `bibliotheek.json` op te halen
- De response omzet naar JSON
- De boeken afdrukt op een nette manier
- Afhandelingen gebeuren in een `try/catch` blok
- Het resultaat wordt getoond in het `bibliotheek-result` element

```javascript
const haalBibliotheekData = async () => {
  try {
    // Verwerk en toon de data
  } catch (error) {
    // Foutafhandeling
  }
};
```

#### 6. Event listeners koppelen

Koppel beide functies aan de knoppen:

- De `btn-starwars` knop roept `haalStarWarsData()` op
- De `btn-bibliotheek` knop roept `haalBibliotheekData()` op

Zorg ervoor dat je DOM-elementen goed vindt en events juist implementeert.

#### 7. Toon de data netjes

- Voor Star Wars data: toon naam, bevolking, klimaat, terrein als een nette HTML structuur
- Voor bibliotheek data: toon alle boeken uit de array in een lijst of tabel met titel, auteur, jaar en beschikbaarheid
- Zorg voor **foutafhandeling** op beide manieren en toon een duidelijke foutmelding

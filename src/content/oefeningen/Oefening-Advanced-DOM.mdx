## Oefening: Interactieve Todo-List

In deze oefening oefen je de concepten uit de hoofdstukken **DOM-Tree** en **Events**. Je gaat een interactieve todo-list bouwen waarbij gebruikers kunnen:

- Items toevoegen aan een lijst
- Items als voltooid markeren
- Items verwijderen uit de lijst
- Het aantal voltooide items zien

## Stap 1: HTML-structuur maken

Maak een bestand `todo.html` met volgende HTML-structuur:

```html
<!DOCTYPE html>
<html lang="nl">

    <meta charset="UTF-8" />

        <li class="todo-item">
          <input type="checkbox" class="checkbox" checked />
          <span class="todo-text">Boeken lezen</span>
          <button class="delete-btn">🗑</button>
        </li>
        <li class="todo-item">
          <input type="checkbox" class="checkbox" />
          <span class="todo-text">Sporten</span>
          <button class="delete-btn">🗑</button>
        </li>
      </ul>

      <div class="input-group">
        <input
          type="text"
          id="newTodoInput"
          placeholder="Voeg een nieuw item toe..."
        />
        <button id="addBtn">Toevoegen</button>
      </div>

      <p id="counter"></p>
    </div>

    <script src="main.js"></script>
  </body>
</html>
```

## Stap 3: JavaScript-interactiviteit

Maak een bestand `main.js` en implementeer daarin de volgende functies:

### Vereisten:

1. **Setup-functie**: Een `window load` event dat een `setup()` functie aanroept
2. **Elementen opvragen**: Gebruik `getElementById()` en `getElementsByClassName()` om alle nodige HTML-elementen op te halen
3. **Counter updaten**: Als een checkbox wordt aangeklikt, update het aantal voltooide items
4. **Item toevoegen**: Bij het klikken op de "Toevoegen"-knop:
   - Lees de input-waarde uit
   - Maak een nieuw list item aan
   - Voeg dit item toe aan de DOM
   - Clear de input-veld
   - Update de counter
5. **Item verwijderen**: Bij het klikken op de verwijder-knop, verwijder het item en update de counter
6. **Checkbox styling**: Wanneer een checkbox wordt aangeklikt, voeg de `completed` class toe aan het li-element

### Tips:

- Gebruik `addEventListener()` voor het toevoegen van event listeners
- Gebruik `classList.toggle()` of `classList.add()` voor het toevoegen/verwijderen van classes
- Zet event listeners op alle delete-buttons en checkboxes
- Gebruik `textContent` of `innerHTML` om nieuwe elementen te maken met `createElement()` en `appendChild()`
- De counter moet het format `"Voltooide items: X/Y"` hebben, waar X het aantal voltooide items is en Y het totale aantal

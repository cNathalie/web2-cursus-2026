## Voorbeeldoplossing: Interactieve Todo-List

Dit is een complete werkende voorbeeldoplossing voor de oefening "Interactieve Todo-List". Bestudeer goed hoe de concepten uit DOM-Tree en Events hier toegepast worden.

---

## JavaScript (`main.js`)

```javascript
// ========================================
// Setup functie
// ========================================
// Dit is de functie die wordt uitgevoerd wanneer
// de pagina volledig is geladen.
const setup = () => {
  console.log("Pagina geladen, setup gestart!");

  // Haal alle nodige elementen op uit de DOM
  // Hier gebruiken we getElementById() zoals uit
  // het hoofdstuk "Elementen opvragen" geleerd
  const addBtn = document.getElementById("addBtn");

  // Haal alle checkboxes op met getElementsByClassName()
  // Dit geeft een HTMLCollection terug
  const checkboxes = document.getElementsByClassName("checkbox");

  // Haal alle delete-buttons op
  const deleteButtons = document.getElementsByClassName("delete-btn");

  // ========================================
  // Event listeners instellen
  // ========================================

  // Voeg event listeners toe aan alle bestaande checkboxes en buttons
  addEventListeners(checkboxes, deleteButtons);

  // Voeg een event listener toe aan de "Toevoegen" knop
  addBtn.addEventListener("click", handleAddTodo);

  // ========================================
  // Initiële setup
  // ========================================

  // Update de counter bij het laden van de pagina
  // (omdat het tweede item bij het starten al aangevinkt is)
  updateCounter();

  console.log("Setup voltooid!");

  // Bonus: Maak het mogelijk om met Enter een item toe te voegen
  const newTodoInput = document.getElementById("newTodoInput");
  newTodoInput.addEventListener("keypress", function (event) {
    // Check of de toets die ingedrukt werd Enter is
    if (event.key === "Enter") {
      handleAddTodo();
    }
  });
};

// ========================================
// Window load event (BELANGRIJK!)
// ========================================
// We wachten tot het window load event plaatsvindt
// zodat alle HTML-elementen volledig geladen zijn
// voordat we JavaScript code uitvoeren.
window.addEventListener("load", setup);

// ========================================
// Functie: Event listeners toevoegen
// ========================================
// Deze functie voegt event listeners toe aan
// checkboxes en delete-buttons
const addEventListeners = (checkboxList, buttonList) => {
  // Loop door alle checkboxes
  for (let i = 0; i < checkboxList.length; i++) {
    const checkbox = checkboxList[i];

    // Voeg een click event listener toe
    // Als de checkbox wordt geklikt, voer handleCheckbox uit
    checkbox.addEventListener("click", function () {
      handleCheckbox(this);
    });
  }

  // Loop door alle delete-buttons
  for (let i = 0; i < buttonList.length; i++) {
    const deleteBtn = buttonList[i];

    // Voeg een click event listener toe
    deleteBtn.addEventListener("click", function () {
      handleDelete(this);
    });
  }
};

// ========================================
// Functie: Counter updaten
// ========================================
// Deze functie telt hoeveel items voltooid zijn
// en update de weergave
const updateCounter = () => {
  const counterDisplay = document.getElementById("counter");

  // Haal alle items op
  const checkboxes = document.getElementsByClassName("checkbox");

  // Tel hoeveel items de class "completed" hebben
  let completedCount = 0;

  // We loopen door alle items (dit is een andere manier
  // dan getElementsByClassName, maar werkt hetzelfde)
  for (let i = 0; i < checkboxes.length; i++) {
    if (checkboxes[i].checked) {
      completedCount++;
    }
  }

  // Tel totaal aantal items
  const totalCount = checkboxes.length;

  // Update de tekst in de counter
  // Dit gebruiken we textContent (zoals geleerd in het hoofdstuk)
  counterDisplay.textContent = `Voltooide items: ${completedCount}/${totalCount}`;
};

// ========================================
// Functie: Nieuw item toevoegen
// ========================================
const handleAddTodo = () => {
  const newTodoInput = document.getElementById("newTodoInput");
  const todoList = document.getElementById("todoList");

  // Haal de waarde uit het input-veld
  // Hier gebruiken we value property van het input element
  const newTodoText = newTodoInput.value.trim();

  // Check of het input-veld niet leeg is
  if (newTodoText === "") {
    alert("Voer alsjeblieft een item in!");
    return;
  }

  // Maak een nieuw <li> element aan
  // createElement() is een DOM-method om elementen aan te maken
  const newLi = document.createElement("li");
  newLi.className = "todo-item";

  const newInput = document.createElement("input");
  newInput.type = "checkbox";
  newInput.classList.add("checkbox");

  const newSpan = document.createElement("span");
  newSpan.classList.add("todo-text");
  newSpan.textContent = newTodoText;

  const newBtn = document.createElement("button");
  newBtn.classList.add("delete-btn");
  newBtn.innerText = "X";

  newLi.appendChild(newInput);
  newLi.appendChild(newSpan);
  newLi.appendChild(newBtn);

  // We kunnen dit in principe ook doen met innerHTML in te vullen met een HTML string,
  // maar dit is een duurdere operatie. We verkiezen het aanmaken van nieuwe elementen
  // zoals hierboven.
  // newLi.innerHTML = `
  //         <input type="checkbox" class="checkbox">
  //         <span class="todo-text">${newTodoText}</span>
  //         <button class="delete-btn">X</button>
  //     `;

  // Voeg het nieuwe element toe aan de todo-list
  // appendChild() voegt een child-node toe aan het einde
  todoList.appendChild(newLi);

  // Clear het input-veld
  newTodoInput.value = "";

  // Haal de checkbox en delete-button van het nieuw toegevoegde element op
  const newCheckbox = newLi.querySelector(".checkbox");
  const newDeleteBtn = newLi.querySelector(".delete-btn");

  // Voeg event listeners toe aan het nieuwe element
  newCheckbox.addEventListener("click", function () {
    handleCheckbox(this);
  });

  newDeleteBtn.addEventListener("click", function () {
    handleDelete(this);
  });

  // Update de counter
  updateCounter();
};

// ========================================
// Functie: Checkbox handler
// ========================================
// Deze functie wordt uitgevoerd wanneer
// op een checkbox wordt geklikt

const handleCheckbox = (checkbox) => {
  // Haal het parent <li> element op
  // Hier gebruiken we parentElement (family-tree relatie)
  const todoItem = checkbox.parentElement;

  // Toggle de "completed" class
  // Dit voegt de class toe als hij er niet is,
  // en verwijdert hem als hij er wel is
  if (checkbox.checked) {
    todoItem.classList.toggle("completed");
  }

  // Update de counter
  updateCounter();
};

// ========================================
// Functie: Delete handler
// ========================================
// Deze functie wordt uitgevoerd wanneer
// op de delete-button wordt geklikt

const handleDelete = (button) => {
  // Haal het parent <li> element op
  const todoItem = button.parentElement;

  // Verwijder het element uit de DOM
  // removeChild() verwijdert een child-node
  todoItem.parentElement.removeChild(todoItem);

  // Update de counter
  updateCounter();
};

```

## Stap-voor-stap uitleg van de code

### 1. **Window Load Event**

```javascript
window.addEventListener("load", setup);
```

Dit zorgt dat onze `setup` functie pas wordt uitgevoerd als de pagina volledig geladen is. Dit is belangrijk omdat we elementen uit de DOM moeten ophalen.

### 2. **Elementen opvragen (DOM-Tree)**

```javascript
const todoList = document.getElementById("todoList");
const checkboxes = document.getElementsByClassName("checkbox");
```

Hier gebruiken we de methodes uit het hoofdstuk "Elementen opvragen" om HTML-elementen op te halen.

### 3. **Event Listeners toevoegen (Events)**

```javascript
checkbox.addEventListener("click", function () {
  handleCheckbox(this);
});
```

We voegen event listeners toe aan elementen. Wanneer een event plaatsvindt (click), wordt de functie uitgevoerd.

### 4. **DOM manipulatie**

```javascript
todoItem.classList.toggle("completed");
```

Met `classList` kunnen we classes toevoegen en verwijderen. Dit past de styling aan.

### 5. **Event Handler Functions**

`handleCheckbox()` en `handleDelete()` zijn functies die worden uitgevoerd wanneer bepaalde events plaatsvinden.

## Belangrijke concepten uit DOM-Tree en Events

| Concept              | Voorbeeld in code                     |
| -------------------- | ------------------------------------- |
| Document opvragen    | `document.getElementById()`           |
| Element Node         | `<li>` elementen                      |
| Parent-Child relatie | `checkbox.parentElement`              |
| Event Listener       | `addEventListener("click", ...)`      |
| Event Handler        | `handleCheckbox()` functie            |
| DOM manipulatie      | `appendChild()`, `classList.toggle()` |
| Element ophalen      | `document.getElementsByClassName()`   |

## Veelgemaakte fouten om te vermijden

### ❌ **Fout 1: Geen window load event**

```javascript
// FOUT - elementen zijn nog niet geladen!
const todoList = document.getElementById("todoList");
```

### **Correct**

```javascript

const setup = () => {
  const todoList = document.getElementById("todoList");
  // Nu zijn alle elementen geladen
};

window.addEventListener("load", setup);
```

---

### ❌ **Fout 2: Event listeners niet toevoegen aan nieuwe elementen**

```javascript
// FOUT - het nieuwe element krijgt geen event listener
const newLi = document.createElement("li");
dialogList.appendChild(newLi);
// De checkbox in newLi werkt niet!
```

### **Correct**

```javascript
const newLi = document.createElement("li");
todoList.appendChild(newLi);

// Event listeners toevoegen aan het nieuw gemaakte element
const newCheckbox = newLi.querySelector(".checkbox");
newCheckbox.addEventListener("click", handleCheckbox);
```

---

### ❌ **Fout 3: classList method vergeten**

```javascript
// FOUT - classList.add is de juiste method
todoItem.classlist.add("completed"); // classlist bestaat niet!
```

### **Correct**

```javascript
todoItem.classList.add("completed");
// of
todoItem.classList.toggle("completed");
```

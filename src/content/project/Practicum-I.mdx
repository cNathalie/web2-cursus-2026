## Set up

Indien je dit nog niet gedaan hebt, installeer alle nodige programma's:

- Visual Studio Code
- GitHub Desktop

Zorg ervoor dat je een actieve GitHub studentenaccount hebt en de Classroom assignment hebt aanvaard (zie 'GitHub Classroom') .

Voer alle stappen uit die staan beschreven in 'Startpunt'.

## Praktische opdracht: jouw project starten

Voor dit eerste practicum ga je écht van start met je eigen project.
Dit is nog maar het begin: alles wat je nu maakt (**behalve het gekozen onderwerp**) mag en zal je later nog verder uitbreiden en verbeteren.

#### 1. Bestudeer het project

Het startpunt van je project is gemaakt met Vite en bevat dus al een mini-website. Bekijk hoe alle pagina's (html, css, js)
aan elkaar gelinkt zijn en elkaar beïnvloeden. Er staat ook al een voorsmaakje JavaScript code in waaruit je kan zien hoe je een pagina interactief kan maken.

Merk op hoe de mappenstructuur een duidelijk overzicht geeft van je project. Het is belangrijk dat je deze structuur altijd respecteert.

#### 2. Kies je onderwerp

Bedenk een onderwerp voor je website. Kies iets dat je leuk vindt en waar je graag rond werkt.
Inspiratie vind je in de pagina 'Onderwerp'.

#### 3. Maak een minimalistische homepagina

Open het bestand `index.html` in de root van je project. Maak hier een eenvoudige startpagina met:

- Een duidelijke titel van je project (het onderwerp)
- Een introductie van het onderwerp
- Eén afbeelding die past bij je onderwerp

De titel moet gestyled zijn via een gelinkte CSS-file (bijvoorbeeld door een kleur, lettertype of grootte aan te passen in een CSS-bestand in de map `src/styles`).

> Je mag alles later nog aanpassen en uitbreiden! Dit is enkel een eerste versie om je project op gang te trekken.

#### 4. Commit je werk naar GitHub

1. Sla je wijzigingen op in Visual Studio Code.
2. Commit je aanpassingen via GitHub Desktop (zie de handleiding 'GitHub Desktop').
3. Push je commit naar GitHub zodat je werk ook online bewaard is.

---

Gefeliciteerd, je hebt nu de eerste stap gezet naar je eigen webproject! In de volgende lessen en practica ga je deze basis verder uitbouwen en verbeteren.

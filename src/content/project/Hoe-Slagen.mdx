# Must have features in je webapplicatie
De volgende zaken **moeten _allemaal_** aanwezig zijn in je project om te kunnen slagen:

- Juiste bestandsstructuur met aparte mappen voor HTML, CSS en JavaScript bestanden
- Minimaal 5 verschillende HTML pagina's die je kan bereiken via een navigatiebalk.
- Elke pagina moet een unieke titel hebben die in de browser tab wordt weergegeven.
- Minimaal 3 verschillende CSS stijlen die je toepast op je HTML elementen.
- Volgende JavaScript functionaliteiten die interactie toevoegen aan je webpagina's:
    - Knop(pen) die bij een klik een actie uitvoeren (bijv. tekst veranderen, afbeelding wisselen, achtergrondkleur aanpassen).
    - Formulier(en) die gebruikersinvoer accepteren en valideren voordat deze wordt "verzonden" (formulieren moeten **niet** worden verzonden naar een back-end systeem).
    - Dynamisch toevoegen of verwijderen van HTML elementen op de pagina via JavaScript.
    - Gebruik van ten minste één array om gegevens op te slaan en te manipuleren (bijv. sorteren).
    - Gebruik van lokale opslag (localStorage) om gegevens op te slaan.
    - Gebruik van ten minste één custom data-attribuut om extra informatie aan HTML elementen toe te voegen en deze informatie te gebruiken in je JavaScript code.
    - Gebruik van ten minste één string methode om tekst te manipuleren.
    - Gebruik van ten minste één timer functie( bijv. voor een animatie).
    - Gebruik van event listeners om gebruikersinteracties af te handelen.
    - Gebruik van functies om herbruikbare code te schrijven.
    - Gebruik van node-navigatie om door de DOM te traverseren (bijv. parentNode, childNodes, nextSibling).
    - Gebruik van conditionele logica (if/else statements) om beslissingen te nemen op basis van gebruikersinvoer of andere voorwaarden.
    - Gebruik van CSS-selectoren om HTML elementen te selecteren en te manipuleren.
    - Het bezoeken van de webapplicatie moet ervoor zorgen dat er een Cookie met een levensduur wordt ingesteld in de browser.
    - Gebruik van de Fetch API om gegevens op te halen van een externe bron (bijv. een JSON bestand).

# PowerPoint presentatie
Je moet een korte PowerPoint presentatie geven (maximaal 5 slides) waarin je de volgende zaken bespreekt:
- Je project idee/onderwerp en doelstellingen.
- Overzicht van je project en de functionaliteiten die je hebt geïmplementeerd.
- Technische uitdagingen die je bent tegengekomen en hoe je deze hebt opgelost.
- Toekomstige verbeteringen of uitbreidingen die je zou willen doorvoeren.

# Demo
Na je presentatie zal je jouw project lokaal opstarten en een live demo geven van de functionaliteiten die je implementeerde.

# Q&A
Hierna volgt een Q&A sessie waarin je vragen van de docent kunt beantwoorden.

# Beoordelingscriteria
- Je hebt alle 15 gevraagde JavaScript functionaliteiten geïmplementeerd.
    - Elk van de vereiste JavaScript functionaliteiten werkt correct.
    - Het volledig ontbreken van één van de 15 gevraagde JavaScript functionaliteiten resulteert in een onvoldoende beoordeling voor dat onderdeel.
- Bestandsstructuur en technische organisatie
    - Je project heeft de juiste bestandsstructuur.
    - Je project heeft minstens 5 HTML paginas en duidelijke navigatie.
    - Unieke titels per pagina.
    - Minimaal 3 verschillende CSS stijlen toegepast.
- Presentatie en demo
    - Structuur en duidelijke presentatie
    - Uitleg over project en technische keuzes
    - Demo toont alle kernfuncties
- Soft Skills
    - Vlot en duidelijk spreken
    - Zelfzeker en op een correcte manier antwoord geven op vragen
    - Professionele houding en tijdsbeheer

### **Formulier validatie** (als je nog geen formulier hebt)

Voeg formulier validatie toe aan een formulier op een van je pagina's:

```javascript
form.addEventListener("submit", function(event) {
    event.preventDefault();
    
    // Haal waarden op
    const inputValue = document.getElementById("input").value.trim();
    
    // Valideer
    if (inputValue === "" || inputValue.length < 3) {
        console.log("Input ongeldig");
        return;
    }
    
    // Verwerk data
    console.log("Formulier ingevuld:", inputValue);
});
```

**Tips:**
- Zorg voor duidelijke foutmeldingen bij ongeldige input
- Gebruik `.trim()` om spaties aan begin/einde te verwijderen
- Te kort? Gebruik `.length` om aantal karakters te checken

### **Dynamische elementen (createElement, appendChild)**

Voeg de mogelijkheid toe iets aan je pagina toe te voegen/verwijderen. Dit kan alles zijn passend bij je onderwerp:

```javascript
const createButton = document.getElementById("create");
const container = document.getElementById("container");

createButton.addEventListener("click", function() {
    // Maak nieuw element
    const newElement = document.createElement("div");
    newElement.textContent = "Ik ben net gemaakt!";
    newElement.className = "new-item";
    
    // Voeg toe aan pagina
    container.appendChild(newElement);
});
```

**Voorbeelden per onderwerp:**
- **E-shop:** Voeg product naar winkelwagen toe
- **Blog:** Voeg commentaar toe aan bericht
- **Quiz:** Voeg vraag toe wanneer antwoord goed is
- **Portfolio:** Voeg project toe aan galerij
- **Forum:** Voeg post/reply toe

### **Arrays voor data**

Sla data op in een array en manipuleer deze:

```javascript
// Data opslaan in array
let items = [];

function addItem(itemText) {
    items.push(itemText); // Voeg toe
}

function removeItem(index) {
    items.splice(index, 1); // Verwijder 1 element
}

function getAllItems() {
    return items; // Haal alles op
}

// Array methodes
items.filter(item => item.includes("belangrijk")); // Filter
items.sort(); // Sorteer
items.length; // Lengte
```

**Voorbeelden:**
- Array van producten, gebruikers, berichten, likes, etc.
- Sorteer array (A-Z, nieuwste eerst, prijzen oplopend, etc.)
- Filter array (alleen actieve items, alleen goedkoop, etc.)

### **localStorage - Persistent data**

Sla gebruikersdata op zodat het blijft na page refresh:

```javascript
// Sla data op
function saveData(key, value) {
    localStorage.setItem(key, value);
}

// Haal data op
function getData(key) {
    return localStorage.getItem(key);
}

// Voorbeelden
saveData("userName", "Jan");
saveData("theme", "dark");

const name = getData("userName"); // "Jan"
```

**Voor complexe data, gebruik JSON:**

```javascript
// Object opslaan als JSON
const user = { name: "Jan", email: "jan@example.com" };
localStorage.setItem("user", JSON.stringify(user));

// JSON teruglezen
const savedUser = JSON.parse(localStorage.getItem("user"));
```

**Mogelijkheden:**
- Gebruikersnaam onthouden
- Instellingen (theme, taal, etc.)
- Favorieten opslaan
- Progress tracking
- Winkelwagen opslaan
- Boodschappenlijstje

### **Custom data-attributen**

Voeg extra informatie aan HTML elementen toe:

```html
<!-- HTML -->
<div class="product" data-id="123" data-price="29.99">
    Mijn product
</div>
```

```javascript
// JavaScript - data ophalen
const product = document.querySelector(".product");
const id = product.getAttribute("data-id");      // "123"
const price = product.dataset.price;              // "29.99"

// Data gebruiken
if (id === "123") {
    console.log("Dit is het juiste product!");
}
```

**Gebruikerscases:**
- Product-IDs, gebruikers-IDs opslaan
- Prices, categories opslaan
- Statuses, states tracken
- Metadata aan elementen toevoegen

### **String methodes**

Manipuleer tekst in JavaScript:

```javascript
const text = "  Hello World  ";

text.trim();           // "Hello World" - removes spaces
text.includes("World"); // true - check if contains
text.split(" ");        // ["Hello", "World"] - splits in array
text.replace("World", "JavaScript"); // "  Hello JavaScript  "
text.toUpperCase();     // "  HELLO WORLD  "
text.toLowerCase();     // "  hello world  "
text.substring(0, 5);   // "  Hell" - get part

// Combineert
const cleanedText = text.trim().toLowerCase();
```

**Praktische toepassingen:**
- Input validatie (lengte checken, spaties verwijderen)
- Zoeken (includes)
- Filtren (split, includes)
- Formatting (toUpperCase, substring)

### **Timers (setTimeout, setInterval)**

Voer code uit na vertraging of regelmatig:

```javascript
// setTimeout - eenmalig na vertraging (in milliseconden)
setTimeout(function() {
    console.log("Dit gebeurt na 2 seconden");
}, 2000);

// setInterval - herhaal om de X milliseconden
const intervalId = setInterval(function() {
    console.log("Dit herhaalt elke seconde");
}, 1000);

// Stop de interval
clearInterval(intervalId);
```

**Voorbeelden:**
- Auto-verbergen van meldingen
- Aftelklok (countdown timer)
- Slideshow met fotos
- Pulserende animatie
- Refresh gegevens om de X seconden
- Flitsende waarschuwing

### **Node navigatie**

Navigeer door de DOM-tree:

```javascript
// HTML structuur
// <ul id="list">
//   <li class="item" id="item1">Item 1</li>  // target
//   <li class="item" id="item2">Item 2</li>
// </ul>

const target = document.getElementById("item1");

// Parent
const parent = target.parentNode; // <ul>

// Siblings
const nextSibling = target.nextElementSibling; // item2
const prevSibling = target.previousElementSibling; // null

// Children
const parentChildren = parent.children; // [item1, item2]
const firstChild = parent.firstElementChild; // item1
```

**Wanneer nodig:**
- Verwijder een element via `parentNode.removeChild()`
- Voeg element in naast ander element
- Zoek omhoog naar parent met bepaalde class

### **Cookies met levensduur**

Stel cookies in die blijven (anders dan sessionStorage):

```javascript
// Cookie instellen (dias = days)
function setCookie(name, value, days) {
    const date = new Date();
    date.setTime(date.getTime() + (days * 24 * 60 * 60 * 1000));
    const expires = "expires=" + date.toUTCString();
    document.cookie = name + "=" + value + ";" + expires + ";path=/";
}

// Cookie lezen
function getCookie(name) {
    const nameEQ = name + "=";
    const cookies = document.cookie.split(";");
    for (let i = 0; i < cookies.length; i++) {
        let cookie = cookies[i].trim();
        if (cookie.indexOf(nameEQ) === 0) {
            return cookie.substring(nameEQ.length);
        }
    }
    return "";
}

// Gebruik
setCookie("userName", "Jan", 30); // 30 days
const storedName = getCookie("userName");
```

**Verschillen van localStorage:**
- localStorage: tot je het handmatig wist
- cookies: vervallen na X tijd
- cookies: verzonden naar server

### **Fetch API - Externe data laden**

Laad data van externe bron (bestand, API):

```javascript
fetch("data.json")
    .then(response => response.json()) // Parse JSON
    .then(data => {
        console.log(data); // gebruikt data
        renderData(data);   // verwerk het
    })
    .catch(error => console.log("Error:", error));
```

**Versimpelde versie:**

```javascript
async function loadData() {
    try {
        const response = await fetch("data.json");
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.log("Error:", error);
    }
}

loadData();
```

**Mogelijke gegevensbronnen:**
- Lokaal JSON bestand (bijv. producten, users, quotes, etc.)
- Openbare API (weer, cryptocurrency, jokes, etc.)
- Je eigen backend server

**Voorbeeld data.json:**
```json
[
    { "id": 1, "name": "Item 1", "beschrijving": "..." },
    { "id": 2, "name": "Item 2", "beschrijving": "..." }
]
```

# Frequently Asked Questions

Heb je verdere vragen over de opdracht?
Aarzel niet en neem contact op via e-mail (nathalie.courteaux@hogent.be) om je vraag te stellen.
Vragen die relevant kunnen zijn voor andere studenten zullen ook hier worden geplaatst, zodat iedereen er toegang toe heeft.

<QAItem
  question="Zijn er specifieke HTML structuren of CSS stijlen die verplicht of verboden zijn in het project?"
  answer="Geen enkele HTML structuren (zoals div's, paragrafen, koppen) of CSS stijlen (zoals kleuren, lettertypes) zijn verplicht noch verboden, zolang je maar minimaal 3 verschillende CSS stijlen toepast en een functionele webapplicatie bouwt."
/>

<QAItem
  question="Is het verplicht om een mobile-first aanpak te volgen bij het bouwen van de webapplicatie?"
  answer="Mobile first is niet verplicht. Deze aanpak is heden ten dage een best practice in webontwikkeling, maar voor deze opdracht is het niet vereist. Je mag dus gerust een desktop-georiënteerde webapplicatie bouwen als dat beter past bij je projectidee. Wie zich confortabel voelt om dit wel te doen, mag natuurlijk ook een mobile-first aanpak volgen. De voorstelling van je project zal op desktop gebeuren, dus zorg ervoor dat je project er ook op desktop goed uitziet en functioneert."
/>

<QAItem
  question="Moeten afbeeldingen in verschillende formaten worden aangeboden voor verschillende schermgroottes?"
  answer="Gebruikte media moeten vlot inlaten, kies dus voor gepaste formaten en optimaliseer afbeeldingen indien nodig."
/>

<QAItem
  question="Hoe zwaar wegen HTML en CSS in de beoordeling?"
  answer="De beoordeling zal voornamelijk gebaseerd zijn op de aanwezigheid en correcte werking van de gevraagde JavaScript functionaliteiten. HTML en CSS worden minder zwaar meegewogen, zolang ze maar functioneel zijn en bijdragen aan een werkende webapplicatie."
/>

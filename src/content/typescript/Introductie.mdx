is dit het perfecte moment om kennis te maken met TypeScript. T
ypeScript is een programmeertaal die bovenop JavaScript is gebouwd. 
Je kunt het zien als een “krachtigere” versie van JavaScript, 
ontworpen om je te helpen beter gestructureerde, foutbestendige code te schrijven.

## Wat is TypeScript?
TypeScript is een superset van JavaScript. 
Dat betekent dat alle geldige JavaScript‑code ook geldige TypeScript‑code is. 
Je hoeft dus niet opnieuw te leren programmeren. TypeScript voegt vooral één belangrijk element toe: **types**.

## Waarom types?
In JavaScript kun je variabelen elk mogelijk type waarde geven: een getal, tekst, functie… Dat is flexibel, maar soms té flexibel. Het kan makkelijk gebeuren dat je per ongeluk een verkeerd type doorgeeft, en zulke fouten ontdek je vaak pas tijdens het uitvoeren van je code.
TypeScript helpt je dit soort fouten vroeger te vinden, doordat je kunt aangeven welk type waarde je verwacht. Bijvoorbeeld:
```typescript
let leeftijd: number = 21;
leeftijd = "twintig"; // Fout: TypeScript waarschuwt je!
```
Het doel van TypeScript is niet om je te beperken, maar om duidelijkheid te geven: jij, je medestudenten en later ook collega‑ontwikkelaars kunnen sneller begrijpen hoe je code werkt.
Voordelen van TypeScript voor beginners:

- Minder fouten: 
TypeScript waarschuwt je wanneer je iets doet dat waarschijnlijk niet klopt.
- Betere code‑voorspelling in editors zoals VS Code, dankzij slimme type‑informatie.
- Overzichtelijkere projecten: je begrijpt beter hoe functies, objecten en modules in elkaar zitten.
- Gemakkelijker opschalen: hoe groter je project, hoe waardevoller TypeScript wordt.

## Hoe werkt het onder de motorkap?
Browsers kunnen geen TypeScript uitvoeren. 
Daarom wordt TypeScript altijd vertaald (gecompileerd) naar JavaScript. 
Het eindresultaat blijft dus gewoon JavaScript dat overal werkt. 
Dit betekent ook dat je stap voor stap TypeScript kunt introduceren zonder je hele project te veranderen.

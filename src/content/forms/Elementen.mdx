Er zijn verschillende input varianten die je kan gebruiken in een invulformulier.

## Text Input

Een tekstveld is een van de meest voorkomende input types. Hier kan de gebruiker tekst invoeren. We halen de waarde van een tekstveld op met de `.value` property.

<div className="inline-dynamic-example">
  <input id="text-input" type="text" placeholder="Voer hier tekst in" />
</div>
```html
<input id="text-input" type="text" placeholder="Voer hier tekst in" />
```

```js
const input = document.querySelector("#text-input");
const inputValue = input.value;
console.log(inputValue);
```

## Radio Buttons

Radio buttons zijn cirkeltjes waar de gebruiker er maar één van kan selecteren binnen een groep.
We halen de waarde van een radio button op met de .checked property.

<div className="inline-dynamic-example">
  <input id="radio-input-1" type="radio" name="radio-group" /> Optie 1
  <input id="radio-input-2" type="radio" name="radio-group" /> Optie 2
</div>

```html
<input id="radio-input-1" type="radio" name="radio-group" /> Optie 1
<input id="radio-input-2" type="radio" name="radio-group" /> Optie 2
```

```js
const radioButtons = document.querySelectorAll('input[name="radio-group"]');
radioButtons.forEach((radio) => {
  radio.addEventListener("change", () => {
    const selectedValue = radio.value;
    console.log(selectedValue);
  });
});
```

## Overige veelgebruikte formulierelementen

Hieronder vind je een beknopte lijst van andere elementen die vaak voorkomen in formulieren:

- `<label>` — label voor een inputveld, verbetert toegankelijkheid
- `<button>` — knop om te verzenden of acties uit te voeren
- `<fieldset>` — groepeert gerelateerde velden binnen een formulier
- `<legend>` — titel/omschrijving voor een fieldset
- `<datalist>` — biedt suggesties bij het invullen van een inputveld
- `<output>` — toont het resultaat van een berekening of actie
- `<file input type="file">` — laat de gebruiker bestanden uploaden
- `<input type="date">`, `<input type="time">`, `<input type="color">`, `<input type="range">`, `<input type="email">`, `<input type="number">`, ... — gespecialiseerde invoervelden voor specifieke soorten data

Deze elementen maken je formulieren gebruiksvriendelijker, toegankelijker en veelzijdiger!

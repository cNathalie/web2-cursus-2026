Attributen zijn stukjes data die we kunnen toevoegen aan HTML elementen. Deze data kunnen we dan gebruiken om elementen te stylen of om ze te manipuleren met JavaScript.

---

## Gekende attributen

Er zijn een aantal attributen die we al kennen, zoals de 'id' en 'class' attributen. Deze attributen kunnen we gebruiken om elementen te identificeren of om ze te stylen met CSS.

---

## Attributen opvragen

We kunnen de attributen van een element opvragen met de `getAttribute()` methode. Deze methode neemt de naam van het attribuut als argument en geeft de waarde van het attribuut terug.

Hier is een voorbeeld van hoe we de `getAttribute()` methode kunnen gebruiken om de waarde van een attribuut op te vragen:

```js
// Haal de titel van deze div op
const element = document.querySelector("#sub");

// Haal de waarde van het 'class' attribuut op
const value = element.getAttribute("class");
console.log(value);
```

Attributen wijzigen
We kunnen de waarde van een attribuut wijzigen met de setAttribute() methode. Deze methode neemt de naam van het attribuut en de nieuwe waarde van het attribuut als argumenten.

Hier is een voorbeeld van hoe we de setAttribute() methode kunnen gebruiken om de waarde van een attribuut te wijzigen:

```js
// Haal de titel van deze div op
const element = document.querySelector("#sub2");

// Wijzig de waarde van het id attribuut
element.setAttribute("id", "new-id");

// Haal de nieuwe waarde van het id attribuut op
const newId = element.getAttribute("id");
console.log(newId);
```

## Aanwezigheid van een attribuut controleren

We kunnen controleren of een element een bepaald attribuut heeft met de hasAttribute() methode. Deze methode neemt de naam van het attribuut als argument en geeft true terug als het attribuut aanwezig is, en false als het attribuut niet aanwezig is.

Hier is een voorbeeld van hoe we de hasAttribute() methode kunnen gebruiken om te controleren of een element een bepaald attribuut heeft:

```js
// Haal de titel van deze div op
const element = document.querySelector("#sub3");

// Controleer of het class attribuut aanwezig is
const hasClass = element.hasAttribute("class");
console.log(hasClass);

// Controleer of het type attribuut aanwezig is
const hasType = element.hasAttribute("type");
console.log(hasType);
```

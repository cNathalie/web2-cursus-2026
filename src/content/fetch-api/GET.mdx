


Met een **GET request** kunnen we data ophalen van een server.  
Dit is veruit de meest gebruikte HTTP‑requestmethode.

---

## Fetch GET

Om een GET request te maken met de Fetch API, geven we een URL door aan `fetch()`.

De functie retourneert een **promise** die een `response` bevat.  
Om hier data uit te halen gebruiken we `.json()`, dat op zijn beurt opnieuw een promise oplevert.

```js
fetch('https://jsonplaceholder.typicode.com/posts/1')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error(error));
```

## Async / Await
Fetch werkt ook uitstekend met async/await.

```js
const fetchData = async () => {
  const response = await fetch('https://stapi.co/api/v1/rest/character?uid=CHMA0000289509');
  const data = await response.json();
  console.log(data);
};

fetchData();
```
### Uitleg
- Async Function: de functie retourneert een Promise.
- Await: wacht tot de fetch‑operatie voltooid is.
- JSON Parsing: response.json() wacht ook asynchroon.
- Logging: de verkregen data wordt getoond in de console.
- Functie Aanroepen: start de asynchrone operatie.

## Error‑handling
Het is belangrijk om errors af te vangen, zodat de applicatie niet vastloopt.

### Via ``.catch()`` na ``.then()``
```js
fetch('https://stapi.co/api/v1/rest/character')
  .then(response => response.json())
  .catch(error => console.error(error));
```
### Via ``try``/``catch`` met ``async``/``await``
```js
const fetchData = async () => {
  try {
    const response = await fetch('https://stapi.co/api/v1/rest/character');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error(error);
  }
};
fetchData();
```
Deze fetch faalt bewust omdat de API een uid (unique identifier) verwacht.
